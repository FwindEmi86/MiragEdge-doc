白名单系统

自动白名单系统​

这是锐界幻境服务器的第一道安全保障，避免未知的危险用户加入服务器
也是群服互通功能的基础，你需要完成绑定才可以加入服务器

绑定方法​

1. 加入QQ玩家交流群：

  • 加入玩家审核群填写问卷
  • 审核通过后加入QQ交流群

2. 进入服务器：

  • 进入服务器注册完成后会弹出下面的绑定提示


服内绑定提示

3. 群内发送绑定信息：

  • 根据 服内绑定提示 获取到的验证码，在玩家交流群中发送 绑定 [验证码]


交流群发送绑定验证码

解绑方法​

  • 单个QQ账号最多可绑定3个游戏账户ID
  • 如果想绑定更多的新账户需要将旧账户进行解绑
